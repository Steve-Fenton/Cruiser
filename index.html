Cruiser

    Cruiser is a CruiseControl.NET Information Radiator that runs in your browser.

    To use Cruiser, just put this "index.html" file in the "...Program Files/CruiseControl.Net/webdashboards" folder on your CI server. You can rename it if you like. All the scripts and images are contained in this one file.

    The following icons are used in this application:

    Display the main view
    Shows this help page
    View the project in the CruiseControl.Net web dashboard (contains build history and failure information)
    Performs a "Force Build" operation to kick off a manual build
    Increase text size
    Decrease text size
    Shows the settings page
    Reloads all projects from the server.
